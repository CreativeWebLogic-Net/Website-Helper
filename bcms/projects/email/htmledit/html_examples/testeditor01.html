This example demonstrates using QWebEditor to replace textarea in a form and it requires PHP enabled server to view submitted data.

1. The simplest way to create an editor:
HtmlEditCreateControlFromObj({
    strElementName:"pagecontent1",    // form element name
    strWidth:"650px"                  // width of editor. default is "100%".
    })

2. Another example demonstrates how to set the initial value and load stylesheets:
HtmlEditCreateControlFromObj({
    strElementName:"pagecontent2",    // form element name
    strWidth:"650px",                 // width of editor
    strValue:"Editor 2",              // set the initial content of editor
    strCssStyle:["../style.css","../style2.css"]        
                                      // load multiple stylesheets into editor
    })

3. This example creates an editor with "simple" class which is defined in htmledit_styles.js:
HtmlEditCreateControlFromObj({
    strElementName:"pagecontent3",    // form element name
    strWidth:"650px",                 // width of editor
    strValue:"Editor 3",              // set the initial content of editor
    className:'simple'                // specifies a class name defined in htmledit_styles.js
    })

4. Another example creates an editor with "example" class which overrides font list, paragraph list and load default stylesheet:
HtmlEditCreateControlFromObj({
    strElementName:"pagecontent4",    // form element name
    strWidth:"650px",                 // width of editor
    strValue:"Editor 4",              // set the initial content of editor
    className:'example'               // specifies a class name defined in htmledit_styles.js
    })

5. This example demonstrates how to set the base href and the height of the editor:
HtmlEditCreateControlFromObj({
    strElementName:"pagecontent5",    // form element name
    strWidth:"650px",                 // width of editor
    strHeight:"300px",                // height of editor
    strValue:"<img src='images/next.gif' /> Editor 5",
    strBaseHref:"http://www.qwebeditor.com/"
                                      // base href of the editor
    })
